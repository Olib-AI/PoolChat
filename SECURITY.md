# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.2.x   | Yes       |
| < 1.0   | No        |

## Reporting a Vulnerability

If you discover a security vulnerability in PoolChat, please report it responsibly.

**Email**: [security@olib.ai](mailto:security@olib.ai)

**Do NOT open a public GitHub issue for security vulnerabilities.**

We will acknowledge your report within 48 hours and work with you to understand and address the issue.

## Priority Areas

PoolChat handles end-to-end encryption for mesh chat communications. Security reports involving any of the following are treated with the highest priority:

- Key exchange
- Message encryption
- History storage

Thank you for helping keep PoolChat and its users safe.
